# 러블리 서울
서울시 실시간 도시데이터 API를 이용한 혼잡도 기반 관광지 정보 제공 서비스<br>
배포 주소 : https://2-project-teal.vercel.app/

서울시에서 선정한 116곳 핫스팟을 카카오 지도에서 확인하고, 시간대별 혼잡도 및 나이 기반 인기도 정보를 조회 가능합니다.

## 기능 일람
* 장소별 혼잡도 조회
* 장소별 가장 한산한 시간대 조회
* 장소 주변 문화행사, 축제 정보 조회
* 사용자 나이 기반 인기있는 장소 추천
* 즐겨찾기

## 기술 스택
* VITE REACT javascript
* Zustand
* React Query
* Vercel
